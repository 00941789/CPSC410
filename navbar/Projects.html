3 3/23/18 NOTE: This (and all following) project requires C++ 11 support. It is already enabled for this application (assuming you are using the GNU toolchain). In general, to enable C++11 support, click on the project name->Properties-> C/C++ Build->Settings->GCC C++ Compiler-> Dialect ->'Language Standard' dropdown. Then choose at least ISO C++11(-std=c++0x). Make sure you include the pthread library under GCC C++ linker->Libraries. Add pthread in the Libraries (-l) pane on the right.
